URL

ex) url : http://opentutorials.org:3000/main?id=HTML&page=12

http - protocol : 통신규약
opentutorials.org - host(domain) : 인터넷에 연결되어 있는 특정 컴퓨터를 가리키는 주소
3000 - port : 한 대의 컴퓨터에 여러 개의 서버가 작동되고 있을 때, 포트 번호를 이용해 특정 서버에 접속
main - path : 컴퓨터의 특정 디렉토리의 어떤 파일인지를 가리킴
id=HTML&page=12 - query string : 웹서버에게 데이터 전달. 값과 값은 &로 구별
